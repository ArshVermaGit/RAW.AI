# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.5.0] - 2026-01-15

### Added

- **Hyper-Premium UI**: Completely redesigned user interface with a modern aesthetic.
- **Ultra Logic Model**: Introduced new model powered by o1-preview for superior humanization.
- **Advanced AI Detector**: Integrated detailed pattern analysis for better detection checks.
- **Style Selector**: New options to tailor humanization tones (Standard, Academic, Creative, etc.).
- **Razorpay Integration**: Secure global payment processing.

### Changed

- **Onboarding Flow**: Enhanced for a more refined and guiding user experience.

## [2.1.0] - 2025-12-10

### Added

- **New Writing Modes**: Added 'Academic' and 'Business' writing modes.

### Changed

- **Performance**: Optimized text processing algorithms, achieving 2x faster results.
- **Mobile Experience**: significantly improved mobile responsiveness and layout.

### Fixed

- Fixed minor formatting bugs in API output.

## [1.5.0] - 2025-11-01

### Added

- **Pro Plan**: Launched subscription with unlimited word count.
- **API Access**: Developer API for integrating RAW.AI into other applications.
- **Google Authentication**: Integrated secure sign-in via Google.
- **Human Score**: Initial release of the text humanization metric.
